About Text2Voice

Text2Voice is a web application that allows users to convert typed text into natural-sounding speech. The app provides customizable options like voice presets, speed, and pitch for a personalized experience.

The application is built using:

  • Python & Flask: Backend server that handles requests and generates audio files.
  • gTTS (Google Text-to-Speech): Converts text to speech on the server side.
  • pydub + ffmpeg: Used to manipulate audio pitch and speed before sending it to the user.
  • HTML, Tailwind CSS & JavaScript: Creates a responsive, interactive frontend with sliders, buttons, and real-time playback using the browser’s SpeechSynthesis API.
  • Browser APIs: Audio playback and live TTS directly in supported browsers.

Users can also download the generated audio and preview different voice presets like male, female, robot, and calm styles.
